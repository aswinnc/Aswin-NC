## Problem Descriptions

### Problem 1

Create a small calculator using a class.
It performs addition, subtraction, multiplication and division.

### Problem 2

With input `a`, print the first `a` odd numbers.
Example: a = 4 → 1, 3, 5, 7

### Problem 3

With input `a`:

* If `a` is odd → print odd numbers up to `a`
* If `a` is even → print odd numbers up to `a - 1`

### Problem 4

Given a list of numbers, count how many numbers are multiples of 1 to 9.

## How to Run

Use the following commands:

```
python Problem-1.py
python Problem-2.py
python Problem-3.py
python Problem-4.py
```
